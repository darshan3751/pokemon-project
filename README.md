# Pokémon API Project

## 📌 Overview
This project fetches Pokémon data from an API using Python.

## 🚀 Features
- Fetch data from API
- Display Pokémon name, height, and weight

## 🛠️ Tech Used
- Python
- Requests library
- Git & GitHub

## ▶️ How to Run
1. Install dependencies:
   pip install requests

2. Run the script:
   python main.py
